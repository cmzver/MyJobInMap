# Кастомные звуки уведомлений

## Добавление звуковых файлов

Для кастомных звуков уведомлений добавьте файлы в папку `app/src/main/res/raw/`:

### Файлы:
- `notification_task.mp3` (или .ogg, .wav) — для обычных заявок
- `notification_emergency.mp3` — для аварийных заявок

### Требования к файлам:
- Формат: MP3, OGG, WAV
- Длительность: до 5 секунд (рекомендуется 1-2 сек)
- Размер: до 100 KB
- Имена файлов: только латиница и подчёркивания, без пробелов

### Пример структуры:
```
app/src/main/res/raw/
├── notification_task.mp3        # Звук для обычных заявок
└── notification_emergency.mp3   # Звук для аварийных заявок
```

## Рекомендации по звукам

### Обычные заявки (`notification_task`)
- Короткий, приятный звук
- Не слишком громкий
- Длительность: 0.5-1 сек

### Аварийные заявки (`notification_emergency`)
- Более настойчивый звук
- Привлекающий внимание
- Длительность: 1-2 сек

## Где взять звуки

1. **Бесплатные ресурсы:**
   - [Freesound.org](https://freesound.org)
   - [Mixkit.co](https://mixkit.co/free-sound-effects/)
   - [Zapsplat.com](https://www.zapsplat.com)

2. **Системные звуки Android:**
   - Можно использовать стандартные звуки Android

## Проверка

После добавления файлов:
1. Пересоберите проект
2. Удалите приложение с устройства (важно! каналы создаются один раз)
3. Переустановите приложение
4. Проверьте настройки уведомлений в системе

## Примечания

- Каналы уведомлений создаются при первом запуске
- Для изменения звука после первой установки нужно:
  - Либо удалить и переустановить приложение
  - Либо изменить звук вручную в настройках Android
