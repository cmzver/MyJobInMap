## Что это за vault

Vault не заменяет документацию проекта. Он нужен как второй мозг команды:

- карты системы
- контекст по подсистемам
- архитектурные решения
- runbook для повторяемых операций
- заметки по активным инициативам

## Текущие фокусы

- Поддержка production-ready Android приложения в `app/`
- Развитие веб-портала в `portal/`
- Поддержка FastAPI backend в `server/`
- Выделение новой мобильной ветки в `mobile-next/`
