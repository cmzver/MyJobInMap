# Project Map

## Продукт

FieldWorker — система управления полевыми заявками с тремя основными поверхностями:

- Android-приложение для исполнителей
- Web portal для диспетчеров и администраторов
- FastAPI backend как единый API и слой бизнес-логики

## Основные каталоги

- `app/` — текущее Android приложение
- `portal/` — React + TypeScript портал
- `server/` — FastAPI backend, тесты, миграции и сервисы
- `mobile-next/` — новая мобильная линия с KMP-структурой
- `docs/` — официальная документация проекта
- `bot/` — Telegram bot
