# Local Development

## Backend

```bash
cd server
make seed
make run-server
make test
```

## Portal

```bash
cd portal
npm install
npm run dev
```

## Android

- Открыть проект в Android Studio
- Для эмулятора backend доступен по `10.0.2.2:8001`
