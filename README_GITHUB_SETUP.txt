🔑 КАК ПОЛУЧИТЬ PERSONAL ACCESS TOKEN
================================================================================

1. Откройте: https://github.com/settings/tokens
2. Нажмите: "Generate new token (classic)"
3. Введите имя: "FieldWorker"
4. Выберите права: ☑️ repo (полный доступ)
5. Нажмите: "Generate token"
6. ⚠️  СКОПИРУЙТЕ СРАЗУ! (больше не сможете увидеть)
7. Используйте вместо пароля при `git push`

Если потеряли токен, создайте новый по той же ссылке выше.

❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
================================================================================

Q: С чего начать?
A: Откройте GITHUB_QUICK_START.md и следуйте инструкциям

Q: У меня возникла ошибка
A: Читайте раздел "Решение типичных проблем" в GITHUB_QUICK_START.md

Q: Я забыл свой логин GitHub
A: Откройте https://github.com/settings/profile (вверху справа)

Q: Где взять Personal Access Token?
A: Читайте раздел выше "КАК ПОЛУЧИТЬ PERSONAL ACCESS TOKEN"

Q: Я случайно создал репозиторий неправильно
A: Удалите его (Settings → Danger Zone → Delete) и создайте заново

Q: Репозиторий создан, но пуст
A: Проверьте, что выполнили `git push -u origin main` без ошибок

Q: Хочу обновить код
A: Используйте: git add . → git commit -m "..." → git push origin main

Q: Как создать ветку разработки?
A: Выполните: git checkout -b develop → git push origin develop

Q: Как удалить репозиторий?
A: На GitHub: Settings → Danger Zone → Delete this repository
